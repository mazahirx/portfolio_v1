# 🎨 Pixel Portfolio v1

> A minimal black & white pixel-themed portfolio built with **React**, **Vite**, and **Tailwind CSS**.  
> This is my first portfolio version — the simple one before the main launch 🚀

---

## 🧠 About
This portfolio showcases my skills, projects, and links in a clean retro-inspired pixel interface.  
Everything is handcrafted to reflect simplicity, focus, and creativity — no unnecessary colors, just pure black & white vibes 🖤🤍

---

## 🧩 Tech Stack
- ⚛️ **React + Vite**
- 🎨 **Tailwind CSS**
- 💾 **React Icons**
- 🖋 **Google Fonts (Press Start 2P)**

---

## 💡 Features
- 🧱 Pixel-inspired black & white theme  
- 📱 Responsive on desktop (mobile support coming later)  
- 🧩 Hover animations and smooth transitions  
- 🔗 Social links: LinkedIn, GitHub, Facebook, Instagram  
- 💻 Projects section showcasing my builds

---

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/<your-username>/portfolio_v1.git

# Navigate to the project folder
cd portfolio_v1

# Install dependencies
npm install

# Run locally
npm run dev
